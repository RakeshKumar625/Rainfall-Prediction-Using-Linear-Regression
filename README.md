# 🌧 Rainfall Prediction Using Linear Regression

## 📌 Project Overview

This project focuses on predicting rainfall (precipitation levels) using historical weather data.
A Linear Regression model is implemented to analyze how different weather features influence precipitation.

The objective is to demonstrate data preprocessing, feature selection, model training, evaluation, and visualization using Python.

---

## 📊 Dataset

Dataset Used: Austin Weather Dataset

Features include:

* Temperature
* Dew Point
* Humidity
* Sea Level Pressure
* Visibility
* Wind Speed
* Precipitation (Target Variable)

---

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Jupyter Notebook

---

## ⚙️ Project Workflow

1. Data Loading
2. Data Cleaning and Preprocessing
3. Handling Missing Values
4. Feature Selection
5. Train-Test Split
6. Model Training using Linear Regression
7. Model Evaluation (R² Score & MSE)
8. Visualization of Predictions

---

## 📈 Model Performance

* R² Score: 0.14
* Mean Squared Error (MSE): 0.058

The model explains approximately 14% of the variance in precipitation levels.
This indicates that rainfall prediction is a complex problem influenced by nonlinear and seasonal factors.
